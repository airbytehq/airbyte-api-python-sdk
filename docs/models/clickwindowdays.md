# ClickWindowDays

Number of days to use as the conversion attribution window for a pin click action.

## Example Usage

```python
from airbyte_api.models import ClickWindowDays

value = ClickWindowDays.ZERO
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `ZERO`     | 0          |
| `ONE`      | 1          |
| `SEVEN`    | 7          |
| `FOURTEEN` | 14         |
| `THIRTY`   | 30         |
| `SIXTY`    | 60         |
