# SourceOracleEnterpriseEncryptionAlgorithm

This parameter defines what encryption algorithm is used.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `AES256`       | AES256         |
| `AES192`       | AES192         |
| `AES128`       | AES128         |
| `THREE_DES168` | 3DES168        |
| `THREE_DES112` | 3DES112        |
| `DES`          | DES            |
