# TimeGranularityType

Granularity of the statistics for metrics per time period. Must be either "DAY" or "MONTH"

## Example Usage

```python
from airbyte_api.models import TimeGranularityType

value = TimeGranularityType.DAY
```


## Values

| Name    | Value   |
| ------- | ------- |
| `DAY`   | DAY     |
| `MONTH` | MONTH   |
