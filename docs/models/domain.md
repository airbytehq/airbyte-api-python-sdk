# Domain

The domain suffix for the Zoho Inventory API based on your data center location (e.g., 'com', 'eu', 'in', etc.)


## Values

| Name     | Value    |
| -------- | -------- |
| `COM`    | com      |
| `IN`     | in       |
| `JP`     | jp       |
| `EU`     | eu       |
| `COM_AU` | com.au   |
| `CA`     | ca       |
| `COM_CN` | com.cn   |
| `SA`     | sa       |
