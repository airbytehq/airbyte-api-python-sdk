# EngagementWindowDays

Number of days to use as the conversion attribution window for an engagement action.

## Example Usage

```python
from airbyte_api.models import EngagementWindowDays

value = EngagementWindowDays.ZERO
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `ZERO`     | 0          |
| `ONE`      | 1          |
| `SEVEN`    | 7          |
| `FOURTEEN` | 14         |
| `THIRTY`   | 30         |
| `SIXTY`    | 60         |
