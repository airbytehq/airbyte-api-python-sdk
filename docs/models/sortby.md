# SortBy

The order to sort the articles in. Possible options: relevancy,
popularity, publishedAt.


## Example Usage

```python
from airbyte_api.models import SortBy

value = SortBy.RELEVANCY
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `RELEVANCY`    | relevancy      |
| `POPULARITY`   | popularity     |
| `PUBLISHED_AT` | publishedAt    |
