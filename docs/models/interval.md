# Interval

Time-series data point interval. Required for intraday endpoints.



## Values

| Name         | Value        |
| ------------ | ------------ |
| `ONEMIN`     | 1min         |
| `FIVEMIN`    | 5min         |
| `FIFTEENMIN` | 15min        |
| `THIRTYMIN`  | 30min        |
| `SIXTYMIN`   | 60min        |
