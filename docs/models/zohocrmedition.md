# ZohoCRMEdition

Choose your Edition of Zoho CRM to determine API Concurrency Limits

## Example Usage

```python
from airbyte_api.models import ZohoCRMEdition

value = ZohoCRMEdition.FREE
```

| `ULTIMATE`     | Ultimate       |
