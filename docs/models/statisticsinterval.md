# StatisticsInterval

Periodicity of statistics returned. it may be daily(P1D), weekly(P1W) or monthly(P1M).

## Example Usage

```python
from airbyte_api.models import StatisticsInterval

value = StatisticsInterval.P1_D
```


## Values

| Name   | Value  |
| ------ | ------ |
| `P1_D` | P1D    |
| `P1_W` | P1W    |
| `P1_M` | P1M    |
