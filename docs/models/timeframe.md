# TimeFrame

For example 1min, 5min, 15min, 30min, 1hour, 4hour

## Example Usage

```python
from airbyte_api.models import TimeFrame

value = TimeFrame.ONEMIN
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `ONEMIN`     | 1min         |
| `FIVEMIN`    | 5min         |
| `FIFTEENMIN` | 15min        |
| `THIRTYMIN`  | 30min        |
| `ONEHOUR`    | 1hour        |
| `FOURHOUR`   | 4hour        |
