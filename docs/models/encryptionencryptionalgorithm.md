# EncryptionEncryptionAlgorithm

This parameter defines what encryption algorithm is used.

## Example Usage

```python
from airbyte_api.models import EncryptionEncryptionAlgorithm

value = EncryptionEncryptionAlgorithm.AES256
```


## Values

| Name           | Value          |
| -------------- | -------------- |
| `AES256`       | AES256         |
| `RC4_56`       | RC4_56         |
| `THREE_DES168` | 3DES168        |
