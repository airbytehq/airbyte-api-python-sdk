# ViewWindowDays

Number of days to use as the conversion attribution window for a view action.

## Example Usage

```python
from airbyte_api.models import ViewWindowDays

value = ViewWindowDays.ZERO
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `ZERO`     | 0          |
| `ONE`      | 1          |
| `SEVEN`    | 7          |
| `FOURTEEN` | 14         |
| `THIRTY`   | 30         |
| `SIXTY`    | 60         |
