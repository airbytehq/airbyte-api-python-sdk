# MarketNewsCategory

This parameter can be 1 of the following values general, forex, crypto, merger.

## Example Usage

```python
from airbyte_api.models import MarketNewsCategory

value = MarketNewsCategory.GENERAL
```


## Values

| Name      | Value     |
| --------- | --------- |
| `GENERAL` | general   |
| `FOREX`   | forex     |
| `CRYPTO`  | crypto    |
| `MERGER`  | merger    |
