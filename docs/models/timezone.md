# TimeZone

The timezone for the reporting data. Use 'ORTZ' for Organization Time Zone or 'UTC' for Coordinated Universal Time. Default is UTC.

## Example Usage

```python
from airbyte_api.models import TimeZone

value = TimeZone.ORTZ
```


## Values

| Name   | Value  |
| ------ | ------ |
| `ORTZ` | ORTZ   |
| `UTC`  | UTC    |
